## 🚀 Setup & Installation

### Prerequisites
```bash
# No prerequisites needed! Pure HTML/CSS/JS
# Just need a modern web browser
```

### Local Development

1. **Clone the repository**
```bash
git clone https://github.com/kunal-kashyap635/portfolio.git
cd portfolio
```

2. **Open in browser**
```bash
# Simply open index.html in your browser
# Or use a local server:

# Using Python
python -m http.server 8000

# Using Node.js
npx serve

# Using VS Code Live Server
# Right-click on index.html → "Open with Live Server"
```

3. **Access the portfolio**
```
http://localhost:8000
```

## 📧 Contact Form

The contact form is powered by **FormSubmit** - a free form backend service.

### How It Works

1. **User fills the form** → Form data is sent to FormSubmit
2. **FormSubmit processes** → Validates and formats the data
3. **Email delivered** → Message sent to `Kunalkash50@gmail.com`
4. **Redirect back** → User returns to portfolio contact section

### Setup (Already Configured!)

```html
<form action="https://formsubmit.co/Kunalkash50@gmail.com" method="POST">
  <input type="hidden" name="_subject" value="New Portfolio Contact">
  <input type="hidden" name="_captcha" value="false">
  <input type="hidden" name="_template" value="table">
  <input type="hidden" name="_next" value="https://kunal-kashyap635.github.io/portfolio/#contact">
  <!-- Form fields -->
</form>
```

### First-Time Activation

⚠️ **Important**: The first form submission requires email verification!

1. Submit the form for the first time
2. Check email inbox (or spam folder)
3. Click the "Activate Form" link in the email
4. Form is now live and ready to receive messages! ✅
